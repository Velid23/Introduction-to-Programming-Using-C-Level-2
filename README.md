# Introduction-to-Programming-Using-C++-Level-2
Through this course, I transformed C++ into a real power tool, transitioning from just writing code to gaining absolute control, visibility, and deep understanding of memory behavior and execution flow under the hood.

I mastered professional debugging techniques, utilizing breakpoints and watches to inspect memory values step-by-step, effectively shifting my approach from guessing to confidently understanding exactly why my code works.

I gained strong foundations in memory management and pointer mechanics, safely using pointers, references, and dynamic allocation (new / delete) while clearly navigating $Stack$ vs. $Heap$ memory layouts.

I developed production-grade programming skills by mastering advanced functional concepts (overloading, recursion), creating custom C++ libraries, utilizing dynamic containers like vectors and iterators, and implementing robust file handling operations (CRUD) and exception handling.

In this repository, I have compiled and organized the comprehensive notes taken throughout my learning journey, alongside the complete collection of source code and practical exercises implemented in tandem with the instructor.
